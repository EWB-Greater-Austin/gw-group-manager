# Changelog

## [v1.0.0]

Initial release. Self-serve distribution list subscription management for EWB Greater Austin: a public Google Form drives an `onFormSubmit` Apps Script that adds/removes the submitter from `ewbgeneral@ewbgreateraustin.org` and `newsletter@ewbgreateraustin.org` based on which checkboxes they ticked. Idempotent membership reconciliation via Admin SDK Directory.
